How to Upload Your Kovaak's Data


This guide assumes that you have already downloaded Kovaak 2.0: The Meta and followed the standard download process via Steam. If you have not downloaded Kovaak yet, please do so here.

1. Navigate to Aim Tracker

On Logic, navigate to Aim Tracker and select the up-arrow icon to upload your Kovaaks files. When the pop-up appears, select choose files and navigate to the proper folder.


2. Navigate to Your Kovaak Data Folder

Your Kovaak data should be located inside the following file path: [Your steam lib folder]\steamapps\common\FPSAimTrainer\FPSAimTrainer\statscode>


3. Upload Data

If it's your first time using the tool, upload all files in the folder. If you're updating, select as many files as necessary. The tool will automatically filter out scenarios that have already been uploaded so no need to be precise. Select Upload on the pop-up to complete.
